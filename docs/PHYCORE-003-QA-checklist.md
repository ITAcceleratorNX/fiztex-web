# PHYCORE-003 — QA checklist (Platform Core Lite)

Чеклист для проверки MVP-флоу админки на mock data.  
Базовый URL: `/admin` · логин: `admin@fiztex.local` / `admin123` (или `+77001112233` / `admin123`).  
Без backend тоже должно работать (mock-auth fallback).

**Проверка:** 2026-07-14 — code review + runtime smoke mock-сервисов (`11/11 PASS`).  
UI в браузере вручную не прогонялся (серверы были выключены); логика подтверждена по коду и сервисам.
