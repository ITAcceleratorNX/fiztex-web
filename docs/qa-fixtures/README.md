# QA fixtures для импорта (PHYCORE-003)

Реальный разбор Excel на backend **вне скоупа**.  
UI принимает `.xlsx` / `.xls` / `.csv` и показывает **mock-результат** по выбранному типу импорта.

CSV ниже — тестовые наборы для ручной проверки dropzone и ожидаемых ошибок из mock-слоя.

| Файл | Тип в UI | Ожидаемый mock-результат |
|------|----------|--------------------------|
| `students_ok_with_errors.csv` | Ученики | 18 OK / 2 WARN / 3 ERR (пустое ФИО, дубликат, дата) |
| `parents_with_errors.csv` | Родители | ошибки телефона и дубля |
| `teachers_with_errors.csv` | Учителя | нет контакта, предмет не указан |
| `classes_with_errors.csv` | Классы | дубликат класса, год не найден |
| `invalid.txt` | любой | UI отклоняет до импорта (неверный формат) |

После выбора файла нажмите **Запустить импорт** — имя файла попадёт в шапку результата, строки придут из `MOCK_IMPORT_BY_TYPE`.
